1. Install Python 3.8.2 using terminal or command-line:

For MacOS:
	1. First install Homebrew:
	visit http://brew.sh/.
	2.Press Cmd+C
	3.brew install python3
For Windows:
	1. Visit https://www.python.org/downloads/windows/ and istall Python for your system

2. Install pyenv for Python:
	Visit: https://github.com/pyenv/pyenv
	Example: https://www.youtube.com/watch?v=5DuudrU7jK0&t=115s

3. After you create local enviroment for django project, install django:
	visit https://docs.djangoproject.com/en/3.0/intro/install/

4. Enter the directory yoyr django project and input:
	python manage.py runserver 
	if your see errors visit:
	https://docs.djangoproject.com/en/3.0/intro/tutorial01/
