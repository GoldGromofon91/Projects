==Привет!==
==Это репозиторий сервиса GrowYourself== 
Идея сервиса пришла во время прохождения интерактивного курса по основам HTML/CSS в апреле 2020 года, тогда была создана первая конструкция сайта 

Цель проекта: "GrowYourself" - готовое решение по культивированию домашних 
экзотических фруктов и растений, включающее в себя: 
1)Готовый шкаф с оборудованием
2)Web-сайт*
3)Мобильное приложение(iOS, Android) / Desktop приложение *
4)Система передачи данных
5)Система контроля роста (на основе использования нейронных сетей)

* Сервис организован в виде большого сообщества подключившихся пользователей, с 
* возможностью просмотра потокового видео своих друзей, просмотра online этапов роста, 
* получение рекомендаций и предложений от обученной нейронной сети по выращиванию 

Для дальнейшей реализации такого проекта на 1-м этапе вижу создание  реляциноой БД для хранения данных.
